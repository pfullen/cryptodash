## Crypto dashboard training project

[Online demo](http://cryptodash.cronnection.at/)

## Overview
Training project using:

- create-react-app
- Bitfinex API
- React
- Redux
- moment
- react-highcharts
- styled-components
- cryptocompare


## Functionality

- 2 main pages:
  - dashboard: check the data of your favorite coins
  - Settings: add & remove coins to your favorites list
